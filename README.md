# Exterior Experts CRM

A Jobber-like CRM for Exterior Experts (exteriorexperts.co).

## Apps
- `apps/api` — NestJS REST API (TypeScript)
- `apps/web` — Next.js web frontend (TypeScript)

## Quick Start
1. `docker compose up -d` — starts PostgreSQL
2. `cd apps/api && pnpm dev` — starts API on :3001
3. `cd apps/web && pnpm dev` — starts web on :3000
